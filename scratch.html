Name Project

I used scratch to make my name move around. Some requirements are that each letter had a different sprite and 3 behaviors. When the green flag is clicked, the letters move. For extra credit, I made all of the letters reset to their original position when the green flag is clicked.

Concepts learned

  • Using event blocks
  • Used loops - I used a forever block to make the program continue over and over, I used a repeat until block to make the sprites move smoothly
  • Created multiple sprites
  • Made the sprites move, change size, and change shades.

Computer Buying Project

I interviewed a person and asked them that if they wanted a computer what characteristics the would want the computer to have. After knowing what they would want their computer to be like, my group and I found 4 options for them. After that, my group decided which computer would be the best and made a scratch presentation about it.
